To insert all the data into the database use the following steps

1) Go to PGadmin
2) Create a new database.
3) Open the query tool.
4) Now select all the queries from the create.sql file and paste them in the query tool area.
5) Now run these queries. (All the tables should be created)
6) Now clear the query tool.
7) Copy all the queries from the load.sql file and paste them in the query tool.
8) Run these queries.
9) All the data should be loaded.
